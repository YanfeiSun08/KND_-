# 外圆磨床程序

本项目是专用于KND数控系统的一套外圆磨削解决方案。主要功能有

* 外圆磨削
  * 外圆直切磨削
  * 外圆往复磨削
  * 外圆量仪直切磨削
  * 外圆量仪往复磨削
* 砂轮休整
* 端面量仪
  * 工件端面位置测量
  * 量仪与砂轮间距测量
  * 工件端面误差检查
* 外径量仪
* 自定义UI界面



## 程序结构

* 主程序 O0001 
* 端面测量  ---O9100
  * 
* 工件磨削  ---O9200
  * 
* 砂轮休整  ---OP9300
  *
